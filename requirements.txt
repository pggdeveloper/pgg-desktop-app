# ============================================================================
# Platform-Specific Notes
# ============================================================================
#
# Windows:
#   - All packages should install via pip
#   - RealSense SDK 2.0+ must be installed separately
#   - ZED SDK 4.0+ must be installed separately
#   - Visual C++ Redistributables 2015-2022 required
#
# macOS:
#   - pyobjc packages required for camera access
#   - RealSense SDK must be installed via Homebrew or built from source
#   - ZED SDK must be installed separately
#
# Linux:
#   - pyudev required for device enumeration
#   - RealSense SDK: sudo apt-get install librealsense2-dev
#   - ZED SDK must be installed separately
#
# ============================================================================
# Installation Instructions
# ============================================================================
#
# 1. Create virtual environment:
#    python -m venv .venv
#
# 2. Activate virtual environment:
#    Windows: .venv\Scripts\activate
#    Linux/macOS: source .venv/bin/activate
#
# 3. Install dependencies:
#    pip install -r requirements.txt
#
# 4. Install camera SDKs separately:
#    - Intel RealSense SDK: https://github.com/IntelRealSense/librealsense/releases
#    - ZED SDK: https://www.stereolabs.com/developers/release/
#
